Installing sec-wall on Fedora ™

{% markdown %}
  • Most of the dependencies can be yum-installed {% syntax text %} # yum install python-pip python-gevent python-lxml python-devel libyaml libyaml-devel {% endsyntax %}
  • For the time being, Spring Python, pesto, PyYAML and zdaemon need to be installed via pip {% syntax text %} # pip-python install springpython # pip-python install pesto # pip-python install pyyaml # pip-python install zdaemon {% endsyntax %}
  • Everything's ready for installing sec-wall now {% syntax text %} # pip install sec-wall {% endsyntax %}
  • That's it, head on to the documentation and have fun!
