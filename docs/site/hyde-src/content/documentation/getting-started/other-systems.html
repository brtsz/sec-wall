Installing sec-wall on any system

{% markdown %}
  • sec-wall's written in a highly portable programming language, Python, and its dependencies should allow for anyone to install it on just about any operating system. You just need to make sure you have the necessary packages installed - don't forget that they have their own dependencies in turn. All of the pieces mentioned below can be found at PyPI :
    1. gevent
    2. lxml
    3. pesto
    4. pip
    5. PyYAML
    6. Spring Python
    7. zdaemon
  • Having installed the dependencies, you can now download the sec-wall's .tar.gz package and install it manually with pip.
  • Once you've got it all installed, head on to the documentation and have fun! And hey, don't forget to tell the project where you've installed sec-wall on. Thanks!
