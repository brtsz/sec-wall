Installing sec-wall on Ubuntu

{% markdown %}
  • Most of the dependencies can be apt-installed {% syntax bash %} $ sudo apt-get install python-pip python-pesto python-gevent python-yaml python-lxml {% endsyntax %}
  • For the time being, Spring Python and zdaemon need to be installed via pip {% syntax bash %} $ sudo pip install springpython $ sudo pip install zdaemon {% endsyntax %}
  • Everything's ready for installing sec-wall now {% syntax bash %} $ sudo pip install sec-wall {% endsyntax %}
  • That's it, head on to the documentation and have fun!
